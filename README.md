# 🎓 AI Exam Prep Platform

An intelligent AI-powered learning platform designed to help students prepare smarter, learn faster, and perform better in examinations. The platform combines personalized study planning, AI-assisted tutoring, automated quiz generation, and performance analytics to create a complete academic preparation ecosystem.

---

## 📖 Overview

AI Exam Prep Platform transforms traditional studying into an interactive and personalized learning experience. Students can upload study materials, generate quizzes, interact with an AI tutor, track progress, and receive customized study plans tailored to their academic goals.

Whether preparing for university exams, competitive tests, or technical interviews, the platform provides the tools needed to improve learning efficiency and maximize performance.

---

## ✨ Key Features

### 🤖 AI Tutor

* Interactive AI-powered tutor for instant doubt solving.
* Provides detailed explanations for complex concepts.
* Supports core Computer Science subjects:

  * Data Structures & Algorithms (DSA)
  * Operating Systems (OS)
  * Database Management Systems (DBMS)
  * Computer Networks (CN)
  * Artificial Intelligence (AI)

### 📄 Smart PDF Learning

* Upload lecture notes, textbooks, or study materials.
* Extract important content automatically.
* Generate subject-specific learning resources from uploaded documents.

### 📝 AI Quiz Generator

* Automatically creates quizzes from uploaded PDFs.
* Generates topic-focused and customized assessments.
* Helps students evaluate understanding and identify knowledge gaps.

### 📅 Personalized Study Planner

* Generates daily study schedules based on:

  * Exam date
  * Available study hours
  * Subject priorities
* Creates structured revision plans to improve consistency.

### 📊 Analytics Dashboard

* Visualizes learning progress through interactive charts.
* Tracks study patterns and quiz performance.
* Provides actionable insights to improve academic outcomes.

### 🎯 Progress Tracking

* Monitor completed study sessions.
* Track learning streaks and consistency.
* Measure achievement of daily and weekly goals.

Frontend → React.js

Backend → Node.js + Express.js

Database → MongoDB

AI Engine → Groq AI API

Visualization → Chart.js

Authentication → JWT Authentication

File Processing → PDF Parsing & Analysis
